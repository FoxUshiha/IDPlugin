# IDPlugin
Minecraft Identity Plugin.

Commands:

- /id [Aliases] [Nick] [String]

Permissions:

id.admin (default OP)
id.* [command_name]

Depends of:

Vault.jar https://www.spigotmc.org/resources/vault.34315/download?version=344916
PlaceholderAPI.jar https://www.spigotmc.org/resources/placeholderapi.6245/download?version=613916

Thanks for downloading!
